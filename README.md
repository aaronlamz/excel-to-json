# excel-to-json

Convert Excel files to JSON

## Status
Working in progress.

## Features
### Excel to JSON

```javascript
{
    "i18n": {
        "zhCHT": {
          "KEY_1": "简体中文",
        },
        "zhCHS": {
          "KEY_1": "英文",
        },
        "en": {
          "KEY_1": "繁体中文",
        }
    }
}
```

## Installation

Using npm:

```
$ npm install -g excel-to-json
```

Using yarn:

```
$ yarn global add excel-to-json
```

## Usage

```
etj --sourceFile="example.xlsx" --outputFile="example.json"
```
